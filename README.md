# ✔ 我的待办

一个单文件待办事项管理工具，支持本地存储、多视图、云同步与自动备份。

## 特色

- **单文件架构** — 零依赖，只有一个 `index.html`，打开即用
- **双视图** — 周视图与日历月视图，日历模式下页面自动加宽
- **完整的数据管理** — MD/JSON 导出导入、Gist 云同步、本地定时备份

## 快捷键

| 快捷键 | 功能 |
|---|---|
| `Ctrl + .` | 导出（MD 预览 / JSON 下载 / Gist 上传） |
| `Ctrl + ,` | 导入（粘贴 MD 格式文本） |
| `Ctrl + /` | 打开帮助 |
| `Ctrl + Shift + ?` | 打开设置 |
| `Ctrl + Shift + >` | 从 Gist 下载（需启用 Gist 同步） |

## 使用说明

### 任务管理
- **新增** — 点击底部「➕ 新增任务」按钮
- **编辑** — 双击任务文本，按 `Enter` 保存
- **完成/未完成** — 点击复选框切换
- **删除** — 悬停任务行，点击「删除」按钮
- **调整日期** — 编辑模式下点击「调整日期」按钮
- **拖拽** — 长按拖拽任务到导航栏其他日期或日历格中
- **自动迁移** — 过去的未完成任务会在页面加载时自动转移到今天

### 数据备份
- **本地备份** — 在设置中启用自动备份（JSON 或 MD 格式），可自定义备份间隔
- **Gist 云同步** — 配置 Gist URL 和 Token 后，上传备份或自动定时上传
- **本地文件存储** — 支持通过 File System Access API 选择文件夹直接读写文件（部分浏览器不支持）

### 导出与导入
- `Ctrl + .` 按设定选项在预览标签页中显示 MD 和/或下载 JSON
- `Ctrl + ,` 粘贴符合 MD 格式的待办文本进行导入


## 浏览器支持

Chrome / Edge 推荐。文件系统功能（选择文件夹）需要支持 File System Access API 的浏览器。
