📖 使用说明

📆 基本操作
点击日期切换查看，按 ◀ ▶ 切换上一周/下一周
点击 ➕ 新增任务 创建待办
点击复选框标记完成/未完成
双击待办文本进入编辑，按 Enter 保存
长按拖拽任务到其他日期
过去的未完成任务会自动转移到今天
📅 日历视图
点击 📅 日历视图 切换月视图，自动加宽页面
点击日期格选中该日
双击任务切换完成状态
拖拽任务到其他日期格
日期格右上角 N 显示未完成任务数
💾 数据管理
Ctrl + . 导出 MD/JSON / 上传 Gist（需在设置中启用）
Ctrl + , 导入任务（MD 格式）
Ctrl + Shift + ? 打开设置
Ctrl + / 显示本帮助
在设置中可开启自动备份（本地 JSON/MD 或 Gist 云同步）
状态栏显示当前备份状态
